# sourecetee cli

sourcetree 增加命令行支持

可以设置 alias

```shell
alias sourcetree='open -a SourceTree'

```

然后逕
```shell
SourceTree /open/your/git/repo/path
```

或者直接安装命令行支持, 直接执行

```shell
stree /open/your/git/repo/path
```
